This project is a simple Express.js web application that dynamically displays a random motivational quote, a random Indian recipe, and a random festival detail each time the user submits the form.
Built using Node.js, Express, and EJS templating, the system stores predefined data arrays and renders selected content to the user interface on request.
The project demonstrates handling routes, processing POST requests, passing data to EJS views, and generating dynamic web content efficiently.
Features:
Shows a random motivational quote
Shows a random Indian recipe with ingredients & steps
Shows a random festival with date & description
